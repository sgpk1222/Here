# 绘图与图形管理软件

## 软件简介

这是一款基于 Python + Tkinter 开发的桌面绘图软件，采用**模块化设计**，提供完整的图形绘制、变换和管理功能，适用于课程设计项目。

---

## 模块化架构（便于团队协作）

### 文件结构
```
短学期/
├── main.py                   # 主程序入口（整合所有模块）
├── modules/                  # 功能模块目录
│   ├── __init__.py          # 模块包初始化
│   ├── shapes.py            # 图形类定义模块
│   ├── transforms.py        # 图形变换模块
│   ├── curves.py            # 特殊曲线生成模块
│   ├── file_manager.py      # 文件管理模块
│   └── ui_components.py     # UI界面组件模块
└── README.md                # 说明文档
```

### 各模块职责

| 模块 | 文件 | 功能说明 | 修改建议 |
|------|------|----------|----------|
| **图形类模块** | `shapes.py` | 定义图形数据结构、工厂类、管理器 | 添加新图形类型在此扩展 |
| **变换模块** | `transforms.py` | 图形的移动、缩放、旋转、翻转 | 添加新变换算法在此扩展 |
| **曲线模块** | `curves.py` | 正弦曲线、螺线等特殊曲线生成 | 添加新曲线类型在此扩展 |
| **文件模块** | `file_manager.py` | JSON保存/加载、备份管理 | 文件格式修改在此处理 |
| **UI组件模块** | `ui_components.py` | 工具栏、颜色面板、状态栏等 | 界面调整在此修改 |
| **主程序** | `main.py` | 整合所有模块，协调交互 | 业务逻辑修改在此处理 |

## 功能特性

### 1. 界面设计
- **顶部菜单栏**：文件、编辑、绘图、特殊曲线、变换、创意工具、帮助等菜单
- **左侧工具栏**：绘图工具选择、颜色面板、线宽调节、操作按钮
- **底部状态栏**：实时显示工具状态、鼠标坐标、图形数量
- **中央画布**：白色绘图区域，支持鼠标交互

### 2. 基础绘图功能
| 工具 | 说明 |
|------|------|
| 直线 | 鼠标拖拽绘制直线 |
| 矩形 | 鼠标拖拽绘制矩形 |
| 圆形 | 鼠标拖拽绘制圆形/椭圆 |
| 画笔 | 自由绘制曲线 |
| 橡皮 | 擦除内容 |

### 3. 颜色与线宽
- **颜色选择**：点击颜色框打开系统颜色选择器
- **预设颜色**：10种常用颜色快速选择
- **线宽调节**：1-20像素可调

### 4. 图形变换功能
- **选择**：点击图形进行选择，显示蓝色选择框
- **移动**：选中后拖拽移动图形
- **复制**：Ctrl+C 复制图形（自动偏移）
- **删除**：Delete 键删除选中图形
- **缩放**：菜单选择缩放比例（水平/垂直）
- **旋转**：菜单选择旋转角度（-180°~180°）
- **翻转**：水平翻转、垂直翻转

### 5. 特殊曲线绘制
- **正弦曲线**：可自定义振幅、周期、周期数
- **阿基米德螺线**：可自定义内径参数、螺距参数、圈数

### 6. 图形管理
- **保存**：将画布所有图形保存为 JSON 文件
- **加载**：打开已保存的 JSON 文件恢复图形
- **新建**：清空画布重新开始

### 7. 创意拓展功能
- **彩虹画笔**：绘制彩虹色渐变线条
- **图形填充**：为选中图形添加填充颜色
- **图形阴影**：为选中图形添加阴影效果
- **生成图案**：一键生成圆形图案、星形图案、花瓣图案

## 使用说明

### 绘制基本图形
1. 在左侧工具栏选择绘图工具（直线/矩形/圆形）
2. 选择颜色和线宽
3. 在画布上按住鼠标左键拖拽
4. 释放鼠标完成绘制

### 选择与变换图形
1. 点击工具栏"选择"按钮
2. 在画布上点击要选择的图形
3. 选中后可拖拽移动
4. 通过菜单"变换"进行缩放、旋转、翻转

### 绘制特殊曲线
1. 菜单栏选择"特殊曲线"
2. 选择"正弦曲线"或"阿基米德螺线"
3. 在弹出的对话框中调整参数
4. 点击"绘制"生成曲线

### 保存与加载
- **保存**：菜单"文件" → "保存" → 选择路径 → 保存为 .json 文件
- **加载**：菜单"文件" → "打开" → 选择 .json 文件

### 快捷键
| 快捷键 | 功能 |
|--------|------|
| Ctrl+N | 新建画布 |
| Ctrl+S | 保存文件 |
| Ctrl+O | 打开文件 |
| Ctrl+C | 复制图形 |
| Delete | 删除图形 |
| Esc | 取消选择 |

## 代码结构说明

程序采用模块化设计，主要包含以下模块：

### 模块一：Shape 图形类
- 定义图形基类
- 包含图形类型、坐标点、颜色、线宽等属性
- 提供序列化/反序列化方法

### 模块二：TransformManager 变换管理器
- 实现移动、缩放、旋转、复制等变换操作
- 基于矩阵变换算法

### 模块三：CurveGenerator 曲线生成器
- 正弦曲线生成算法
- 阿基米德螺线生成算法

### 模块四：FileManager 文件管理器
- JSON 格式文件保存/加载
- 图形数据序列化

### 模块五：DrawingApp 主应用类
- 界面创建与管理
- 鼠标事件处理
- 图形绘制与渲染
- 功能菜单实现

## 常见问题

**Q: 为什么图形变形后显示不正常？**
A: 某些极端缩放/旋转可能导致图形超出画布范围，可尝试重新绘制。

**Q: 文件保存后打开显示空白？**
A: 请确保保存时有绘制图形，文件格式为 .json。

**Q: 颜色选择器无法打开？**
A: 请检查系统是否支持颜色对话框，或使用预设颜色。
