# 绘图笔记本 - 书本风格绘图软件

## 软件简介

这是一款基于 Python 3 + Tkinter + Pillow 开发的桌面绘图软件，采用**拟物化书本风格界面**与**高度解耦的模块化设计**，提供完整的图形绘制、仿射变换、特殊数学曲线参数化生成和多页文件管理功能，适用于短学期课程设计与团队分工协作项目。

---

## 模块化架构（便于团队协作）

### 文件结构

```text
Here_test/
├── main.py                   # 极简主程序入口（仅负责启动）
└── modules/                  # 核心功能模块目录
    ├── __init__.py          # 模块包初始化
    ├── app_core.py          # 🧠 主程序中枢模块 (管理逻辑状态、翻页、撤销重做栈)
    ├── ui_builder.py        # 🖼️ UI界面搭建模块 (主界面、菜单栏、2x2背景切换、快速工具栏)
    ├── canvas_handler.py    # 🖱️ 鼠标交互与画笔引擎模块 (含种子填充算法、自适应文本直输)
    ├── math_dialogs.py      # 📐 数学几何配置弹窗模块 (提供所有曲线的高阶参数微调)
    ├── shapes.py            # 图形数据结构模块 (定义各类矢量图形与文本边界判定)
    ├── transforms.py        # 仿射变换模块 (实现图形/喷枪/文本字号跟随移动、缩放与旋转)
    ├── curves.py            # 特殊几何数学公式生成模块
    ├── file_manager.py      # 文件数据序列化保存模块
    └── ui_components.py     # 独立基础UI组件模块
### 各模块职责分工

| 模块名称 | 职责描述 | 对应文件 | 协作修改建议 |
|---------|---------|---------|------------|
| 中枢核心 | 管理全局数据、翻页系统、撤销/重做栈逻辑 | `app_core.py` | 调整核心全局变量在此修改 |
| 界面构建 | 负责拟物化皮肤、菜单栏映射、2x2背景布局 | `ui_builder.py` | 调整工具箱、按钮样式在此修改 |
| 事件控制 | 捕获画布事件、实时绘制预览、种子填充调度 | `canvas_handler.py` | 修改鼠标绘画、拖拽物理反馈在此修改 |
| 数学外脑 | 提供正弦、螺线、极坐标等参数微调弹窗界面 | `math_dialogs.py` | 添加新特殊曲线微调界面在此修改 |
| 矢量模型 | 维护图形属性、序列化接口、自适应文本包围盒 | `shapes.py` | 扩充基础图形实体属性在此修改 |
| 仿射变换 | 基于矩阵公式处理移动、缩放、旋转、镜像算法 | `transforms.py` | 添加全新的仿射变换算法在此扩展 |
| 文件管理 | 处理 JSON 序列化、图像二进制 Base64 无损压缩 | `file_manager.py` | 修改存盘文件格式在此处理 |

## 功能特性

### 1. 界面设计

- **顶部快速工具栏**：提供新建、保存、撤销、重做快捷物理直击按钮，操作无需打开二级菜单。
- **顶部标准菜单栏**：文件、编辑、页面、数学、变换、帮助等经典功能归类。
- **左侧书签工具栏**：自由绘图工具、常用形状横排、12色调色板、笔触与字号调节、自适应背景。
- **2×2 页面背景网格**：彻底消除挤压，支持空白、网格、横线、点阵四种拟物化背景。
- **安全防丢关闭拦截**：检测未保存更改，退出软件时自动拦截确认。

### 2. 基础绘图功能

| 工具 | 说明 |
|------|------|
| 画笔 | 自由绘制平滑矢量曲线 |
| 荧光笔 | 1/4 透明度 stipple 标记，适合做笔记重点 |
| 喷枪 | 喷漆效果，支持墨点群组整体选中与仿射变换 |
| 彩虹 | 随笔触距离动态改变 HSV 颜色的渐变画笔 |
| 文本 | 点击画布生成黄金边框输入框，支持拖拽、确定前修改字号，支持选中后旋转与缩放 |
| 填充 | 采用基于点阵阻骨遮罩的种子漫延填充算法，支持手绘闭合圈及多线段围挡区域填充 |
| 橡皮 | 采用画布背景色擦除内容 |

### 3. 常用形状横排 (类似图2)

- **直线**：鼠标拖拽绘制直线。
- **矩形**：鼠标拖拽绘制矩形，支持填充色。
- **圆形**：鼠标拖拽绘制圆形/椭圆，支持填充色。

### 4. 颜色与属性

- **颜色选择**：点击调色板预设颜色，或点击"自定义颜色"打开系统调色盘。
- **笔触粗细**：1-20像素快速调节。
- **文本字号**：10-72号字大跨度实时调节，并反馈至激活的文本框上。

### 5. 图形仿射变换

- **选择**：选择工具点击图形，生成虚线选择框。
- **移动**：选中后拖拽移动，对喷枪群组和文本提供完美跟随。
- **复制**：`Ctrl+C` 复制选中的矢量对象（自动偏移）。
- **删除**：`Delete` 键删除选中对象（支持精准点选中删除文本）。
- **缩放**：等比例/非等比例缩放（文本字号会同步等比变化）。
- **旋转**：中心点旋转（文本字符会同步发生旋转）。
- **翻转**：水平镜像翻转、垂直镜像翻转。

### 6. 特殊曲线参数化绘制

所有特殊曲线均支持可视化参数调节弹窗：

- **正弦曲线**：自定义振幅、基本周期、周期数（波数）。
- **阿基米德螺线**：自定义起始内径 a、螺距参数 b、圈数。
- **心形曲线**：自定义尺寸比例。
- **利萨如曲线**：自定义水平振幅、垂直振幅、双轴谐振频率。
- **花瓣曲线**：自定义花瓣整体半径、花瓣瓣数。

### 7. 数据管理与导出

- **保存**：将画布及填充层数据编码保存为 `.nb.json` 文件。
- **加载**：载入 `.nb.json` 文件，自动还原多页矢量对象及种子填充图像层。
- **翻页系统**：支持无限新建页面、删除本页，通过 `Ctrl+PageUp/PageDown` 实现翻页。

## 运行环境

### 系统要求

- Python 3.8 或更高版本
- Windows / macOS / Linux

### 依赖库

由于引入了先进的图形种子漫延填充算法（Flood Fill）与图片导入解析，需安装第三方图像处理库：

```bash
pip install Pillow
```

### 运行方法

**方法一：命令行运行**

```bash
python main.py
```

**方法二：IDE运行**

在 VS Code、PyCharm 等集成开发环境中打开根目录，直接运行 `main.py`。

## 使用说明

### 绘制与填充

1. 选择左侧工具栏工具。
2. 拖拽鼠标绘制基础形状、画笔线条。
3. 选用 🪣 填充 工具，点击任何手绘封闭区域或几何图形内部进行种子浸染着色。

### 交互文本录入

1. 点击 🔤 文本 工具，在画布任意位置点击。
2. 在出现的黄金色边框输入框中录入文本。
3. 输入过程中可拖动边框改变其在画布上的位置，或调节左下角滑块改变字号。
4. 按 `Enter` 键或点击画布其他任意区域确认。

## 技术亮点

- **组合模式架构设计**：中枢类作为核心状态机，UI 与鼠标引擎高程度解耦，完全避开循环导入，便于团队并行开发。
- **点阵轮廓屏障扫描技术**：在内存中利用灰度图渲染矢量屏障，打破了矢量图无法进行区域漫延着色的局限。
- **高稳定绝对位移算法**：重构鼠标坐标捕获引擎，文本框拖动彻底告别闪烁、消失与坐标漂移。
- **属性级同步变换机制**：将喷枪墨点集（Scatter Dots）与文本旋转角、文本大小全面整合进放射仿射变换矩阵，保证变换完全一致。
